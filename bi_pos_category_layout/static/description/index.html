Point of Sale Product Category Layout Odoo App

POS Category Grid and List View Layout Odoo Apps

POS Product Multi Categories Grid/List View Layout Odoo Apps

POS Category Grid/List View Layouts odoo app helps users to show point of sale product categories in multiple views like list view or grid view. Users can see category layout in list view or grid view as per configuration. Users can change category layout from list to grid view as per need.

Features

POS Product Category Layout

User can see point of sale product category in different layouts

List or Grid View Category Layout

User can see point of sale product category in list view or grid view as per need.
Demo Request Video Tutorial

POS Category Layout Configuration

Under Point of Sale > Settings menu user need to select "POS Category Layout" option as " List View" to see point of sale product category layout in list view.

List View Category Layout

User can see "&#<" icon in header of pos product screen, on clicking that, category will visible.

User can also hide category column by clicking ">" icon.

Category Layout : List View

User can see all pos product category in list view on point of sale screen.

On clicking specific category user can see that product category and selected category in one column.

POS Category Layout Configuration

Under Point of Sale > Settings menu user need to select "POS Category Layout" option as "Grid View" to see pos category layout in grid view.

View Category Layout

User can see "&#<" icon in header of pos product screen, on clicking that, category will visible.

User can also hide category column by clicking ">" icon.

Category Layout : Grid View

User can see all point of sale product category in grid view on pos screen.

On clicking specific category user can see that product category and selected category in column.
